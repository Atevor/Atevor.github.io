灰色预测模型（Grey Model, GM）

GM(1, 1模型)

pass

GM(2, 1)模型

优缺点分析：

灰色Verhulst预测模型

优缺点：

适用于描述具有饱和状态的过程，即s形过程，例如人口预测、生物生长、繁殖预测、产品经济寿命预测及近年的交通事故发生情况

建立模型：

数据：

  1. 原始数据列：x(0)(k)(k)(k), k = 1, 2, 3 ... n

  2. 累加生成序列：x(1)(k)(k)(k), k = 1, 2, 3 ... n

  3. 均值生成序列：z(1)(k)=(x(1)(k)+x(1)(k−1))∗0.5,k=2...nz^{(1)}(k) = (x^{(1)}(k) + x^{(1)}(k - 1)) * 0.5, k = 2 ... nz(1)(k)=(x(1)(k)+x(1)(k−1))∗0.5,k=2...n

    有 x(1)(k)(k)(k) = z(1)(k)(k)(k) + x(0)(k)(k)(k) / 2 成立

模型：

白化微分方程:

dx(1)(t)dt+ax(t)=b[x(1)(t)]2\frac{dx^{(1)}(t)}{dt}+ ax^{(t)} = b[x^{(1)}(t)]^2dtdx(1)(t)​+ax(t)=b[x(1)(t)]2 t = 0, 1 ... n ①

取近似，得灰微分方程

x(0)(k)+az(1)(k)=b[z(1)(k)]2x^{(0)}(k) + az^{(1)}(k) = b[z^{(1)}(k)]^2x(0)(k)+az(1)(k)=b[z(1)(k)]2 k = 1, 2 ... n ②
将az(1)(k)az^{(1)}(k)az(1)(k) 移至右侧，得矩阵形式 Y=BUY = BUY=BU （自己对应去）

求解：

UUU的最佳观测值采用最小二乘法得到，U=(BTB)−1BTYU = (B^TB)^{-1}B^TYU=(BTB)−1BTY
使用matlab函数工具求得白化微分方程符号解x(1)(t)x^{(1)}(t)x(1)(t), 对应k = t + 1 的 序列值
得到时间响应序列x(1)(k+1)=x(1)(t),t=kx^{(1)}(k + 1) = x^{(1)}(t), t = kx(1)(k+1)=x(1)(t),t=k
累减， 得拟合/预测值 x(0)(k+1)=x(1)(k+1)−x(1)(k),k=1,2...n−1,x(0)(1)=x(1)(1)x^{(0)}(k + 1) = x^{(1)}(k + 1) - x^{(1)}(k), k = 1, 2 ... n-1, x^{(0)}(1) = x^{(1)}(1)x(0)(k+1)=x(1)(k+1)−x(1)(k),k=1,2...n−1,x(0)(1)=x(1)(1)

误差检验

依旧是求残差/相对误差

matlab代码模板：

clc, clear;
%在下面填入数据
x0 = [];                                              %原始序列（这里是行向量）
n = length(x0);
x1 = cumsum(x0);                                      %1次累加（行向量）
z = 0.5 * (x1(2:end) + x1(1:end - 1))'                %均值序列列向量
B = [-z, z.^2];                                            
Y = x0(2:end)';                                       %作数据矩阵/向量
u = B \ Y                                             %最小二乘求解Y = BU

syms x(t);                                            %建立符号函数
x = dsolve(diff(x) + u(1) * x == u(2) * x^2, x(0) == x1(1))      %求解
xt = vpa(x, 6)                                        %控制精度
yuce = subs(x, t, [0:n-1]);                           %将值代入
yuce = double(yuce)                                   %数值解
x0_hat = [yuce(1), diff(yuce)]                        %原始序列的拟似值

epsilon = x0 - x0_hat                                 %残差
delta = abs(epsilon ./ x0)                            %相对误差
